WARNING! WARNING! WARNING!

The mamba and venom builds on this page are built using a modified DTS that increases the kernel partition to 4MB and 6MB respectively.

There is a forum thread about this here

Compatibility check

  • $ fw_printenv | grep "pri_kern_size"; #mamba MUST equal 0x400000
  • $ fw_printenv | grep "priKernSize"; #venom MUST equal 0x0600000
  • Do not try to change them!

Flashing process

  • Accept that this might destroy your box.
  • Be prepared that you might need to recover via serial.
  • Create a backup tar
  • Force flash the corresponding factory image via sysupgrade or stock
  • Restore the backup
  • Success!?
  • Once migrated, you can flash as you normally would.
